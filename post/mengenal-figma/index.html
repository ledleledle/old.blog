Mengenal Aplikasi Figma

 Posted on May 5, 2020  |  1 minutes  |  184 words  |  Leon Prasetya Mulya

Apa itu Figma?

Figma adalah salah satu aplikasi yang digunakan untuk membuat design aplikasi mobile, desktop, website dll.

Kelebihan :

  • Keunggulannya adalah aplikasi ini berbasis web, jadi bisa dibuka di OS apa saja.
  • Tools yang simpel, sedikit dan mudah dipahami.
  • Menyediakan blok code dari design yang telah dibuat (berupa CSS).

Kelemahan :

  • Kelemahannya mungkin karena web based, jadi harus wajib tersambung dengan internet saat penggunaannya.

Untuk menggunakannya caranya mudah, kalian bisa langsung menuju website resminya di figma.com, daftar lalu buat project.

Atau jika kalian pengguna linux, tersedia aplikasi desktopnya (walaupun tidak official). Namun, daripada kita harus membuka browser setiap saat kan. Untuk informasi lebih lengkap tentang aplikasi desktop Figma ini, kalian bisa cek di repository githubnya. Disana juga terdapat tahapan dan cara instalasinya.

Sebelum kalian memulai membuat desain, kalian akan diminta untuk login/daftar akun Figma, kalian login saja dulu, mudah kok. Lalu setelah login tampilan pertama yaitu seperti pada gambar dibawah ini.

Dan tampilan saat membuat desain.

Penutup

Itulah penjelasan dan tutorial dari saya, jika ada kurang lebihnya mohon maaf dan jika ada yang masih bingung bisa menanyakannya di kolom komentar. Sekian dan terimakasih.
